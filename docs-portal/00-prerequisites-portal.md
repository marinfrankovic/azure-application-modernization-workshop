# 00 - Prerequisites - Azure Portal

## Objective

Prepare for the portal-based VM-to-AKS modernization workshop in your own Azure environment.

## Steps

1. Open the Azure portal.
2. Confirm subscription and directory.
3. Confirm you can create resource groups, VNets, VMs, AKS, ACR, APIM, Service Bus, Key Vault, Application Insights, Log Analytics, Defender settings, managed identities, and role assignments as required by your track.
4. Choose a unique prefix and Azure region for your source and destination resources.
5. Run the foundation deployment script to create the source VM and destination VNet.
6. Use the portal for the target build after the foundation exists.

## Expected Outcome

You are ready to create AKS and track-specific target services from the Azure portal.
