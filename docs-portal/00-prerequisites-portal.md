# 00 - Prerequisites - Azure Portal

## Objective

Prepare for the portal-based modernization workshop.

## Steps

1. Open the Azure portal.
2. Confirm subscription and directory.
3. Confirm the facilitator-provided source resource group, source VNet, and source application URL.
4. Confirm you can create resource groups, VNets, Container Apps, AKS, APIM, Service Bus, Key Vault, and monitoring resources.
5. Keep the source environment unchanged unless instructed.

## Expected Outcome

You are ready to create the destination environment from the Azure portal.
