# 00 - Prerequisites - Azure Portal

## Objective

Prepare for the portal-based modernization workshop in your own Azure environment.

## Steps

1. Open the Azure portal.
2. Confirm subscription and directory.
3. Confirm you can create resource groups, VNets, Container Apps, AKS, APIM, Service Bus, Key Vault, and monitoring resources.
4. Choose a unique prefix and Azure region for your source and destination resources.
5. Create or deploy the source environment before creating the destination environment.

## Expected Outcome

You are ready to create both source and destination environments from the Azure portal.
