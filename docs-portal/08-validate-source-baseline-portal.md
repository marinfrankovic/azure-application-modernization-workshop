# 08 - Validate Source Baseline - Azure Portal

## Objective

Confirm your source eShopOnWeb deployment is healthy.

## Steps

1. Open the source application URL.
2. Browse catalog and ordering pages.
3. Open the source VM in the Azure portal.
4. Check VM status, public IP, NSG rules, and boot diagnostics.
5. Save source endpoint details for later comparison.

## Expected Outcome

The source monolith is healthy before destination decomposition begins.
