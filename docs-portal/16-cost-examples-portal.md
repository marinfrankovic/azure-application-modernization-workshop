# 16 - Cost Examples - Azure Portal

## Objective

Estimate and monitor workshop cost from the Azure portal.

## Steps

1. Open Cost Management in the subscription.
2. Filter by workshop resource groups.
3. Set granularity to Daily or Accumulated cost.
4. Compare selected track to sample estimates:
   - Track A 8-hour lab: EUR 2 - EUR 8.
   - Track B 8-hour lab: EUR 18 - EUR 45.
   - Track C 8-hour lab: EUR 12 - EUR 35 before workload compute.
5. Use the formula: hourly rate x running hours + usage charges.
6. Delete both source and destination resource groups when no longer needed.

## Expected Outcome

Cost is visible during the lab and cleanup is tied to running time.
