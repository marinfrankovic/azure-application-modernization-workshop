# 04 - Track A: Container Apps - Azure Portal

## Objective

Create the simple destination environment through the portal.

## Steps

1. Create resource group `rg-appmod-dest-a`.
2. Create destination VNet `appmoda-dest-simple-vnet` with `10.20.0.0/16`.
3. Create a Log Analytics workspace.
4. Create a Container Apps environment attached to the destination VNet.
5. Create catalog, orders, inventory, and notifications container apps.
6. Enable external ingress for catalog and orders only.
7. Test app URLs.

## Expected Outcome

Track A destination is ready for decomposition labs.
