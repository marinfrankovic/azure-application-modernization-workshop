# 03 - Prepare Source Environment - Azure Portal

## Objective

Prepare or inspect the source environment before attendees start.

## Facilitator Steps

1. Create a source resource group.
2. Create a source VNet with address space `10.10.0.0/16`.
3. Deploy the source eShopOnWeb hosting resource.
4. Confirm the source app responds.
5. Share source app URL, source resource group, source VNet, and region with attendees.

## Expected Outcome

The source environment is ready before the workshop begins.
