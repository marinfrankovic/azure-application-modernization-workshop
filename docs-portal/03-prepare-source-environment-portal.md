# 03 - Prepare Source Environment - Azure Portal

## Objective

Prepare the source environment in your own Azure subscription.

## Steps

1. Create a source resource group.
2. Create a source VNet with address space `10.10.0.0/16`.
3. Deploy the source eShopOnWeb hosting resource.
4. Confirm the source app responds.
5. Record source app URL, source resource group, source VNet, and region in your worksheet.

## Expected Outcome

The source environment is ready for your destination track.
