# Parameter Files

Create local parameter files here for workshop runs. Do not commit files containing subscription-specific values, private endpoints, or secrets.

Recommended naming:

- `source.local.parameters.json`
- `track-a.local.parameters.json`
- `track-b.local.parameters.json`
- `track-c.local.parameters.json`

Use the sample file as a starting point and adjust prefix, location, and admin object ID for your environment.
