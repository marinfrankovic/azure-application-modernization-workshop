# Decomposition Strategy

Use eShopOnWeb to identify real modernization boundaries.

## Recommended Order

1. Validate source baseline.
2. Extract catalog reads.
3. Extract notifications behind asynchronous messaging.
4. Extract order intake and inventory reservation.
5. Plan data ownership before production cutover.

## Decision Criteria

- Business capability clarity.
- Data ownership clarity.
- Runtime coupling.
- Rollback path.
- Operational readiness.
- Security and compliance requirements.
